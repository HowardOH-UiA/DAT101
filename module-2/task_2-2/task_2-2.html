  5. What do comparison operators return, and how can we check if they are FALSE or TRUE?

    6. They return a Boolean value, and if the operator is “correct” it returns a true value. An if-statement that does something if the comparison operator is true would tell you whether the operator is right.

  6. How does the Math.random() function work, and what can it be used for?

    7. It gives a random value between a set range. Which is useful whenever something needs to be random.

  7. What is modulus (%), and when do we need this operator in mathematical operations?

    8. The modulus gives what remains after a division is done. Example: 10/3 doesn’t work. 9/3 works. 1 remains and is the modulus. It is necessary when you need to know what remains after an equal division.
